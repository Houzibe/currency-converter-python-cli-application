CURRENCY CONVERTER - Python CLI Application

A simple command-line currency converter that converts Nigerian Naira (NGN) to major world currencies. This project demonstrates fundamental Python concepts including functions, dictionaries, and user input handling.


OVERVIEW

This project was completed as part of a Python programming assignment with the following requirements:


REQUIREMENTS:

✅ Accept user input in Nigerian Naira (NGN)

✅ Convert to at least three currencies (USD, EUR, GBP)

✅ Implement proper function structures

✅ Store exchange rates using dictionaries

✅ Use input() for user interaction

✅ Display clearly formatted output


FEATURES

Convert Nigerian Naira (NGN) to:

- US Dollar (USD)
  
- Euro (EUR)
  
- British Pound (GBP)


TECHNOLOGIES USED

- Python 3.x
  
- No external dependencies required
